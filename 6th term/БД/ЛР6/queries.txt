Список игроков в возрасте до 19 лет и их клубов, сортировать игроков по возрасту===SELECT p.name AS player_name, c.name AS club_name, p.age FROM public.player p INNER JOIN public.club c ON p.club_id = c.id WHERE p.age < 20 ORDER BY p.age;
Список тренеров, которые провели более 100 матчей в лиге, отсортировать по убыванию количества матчей===SELECT t.name AS coach_name, c.name AS club_name, t.games_count FROM public.coach t INNER JOIN public.club c ON t.club_id = c.id WHERE t.games_count > 100 ORDER BY t.games_count DESC;
Список игроков с игровым номером выше 80, которые играют в клубах, находящихся в Москве===SELECT p.name AS player_name, p.number, c.name AS club_name FROM public.player p INNER JOIN public.club c ON p.club_id = c.id WHERE p."number" > 80 AND c.city = 'Москва';
Список команд, у которых количество выигрышей меньше 40 и количество очков больше 80===SELECT c.name AS club_name, cs.wins, cs.ot_wins, cs.pb_wins, cs.points FROM public.club c INNER JOIN public.club_statistics cs ON c.id = cs.club_id WHERE (cs.wins + cs.ot_wins + cs.pb_wins) < 40 AND cs.points > 80;
Список игроков, которые не отличились голами и передачами в сезоне, но заработали штрафные минуты===SELECT p.name AS player_name, pm.goals, pm.assists, pm.penalties FROM public.player p INNER JOIN public.player_match pm ON p.id = pm.player_id WHERE pm.goals = 0 AND pm.assists = 0 AND pm.penalties != 0;
Список белорусских клубов с названием, городом, годом основания и аренами, на которых они играют, с сортировкой по году основания клуба===SELECT c.name AS club_name, c.city, c.foundation_year, a.name AS arena_name FROM public.club c INNER JOIN public.arena a ON c.id = a.club_id WHERE c.country = 'Беларусь' ORDER BY c.foundation_year;
Возраст клубов на основе года основания===SELECT c.name AS club_name, EXTRACT(YEAR FROM CURRENT_DATE) - c.foundation_year AS club_age FROM public.club c ORDER BY club_age DESC;
Список всех арбитров моложе 26  лет и отсортировать по возрасту===SELECT id, name, age, role FROM public.referee WHERE age < 26 ORDER BY age;
Список арен с вместимостью более 6 000 и с годом постройки до 2000 года, отсортированных по вместимости===SELECT id, name, city, construction_year, capacity FROM public.arena WHERE capacity > 6000 AND construction_year < 2000 ORDER BY capacity DESC;
Список матчей, которые ещё не прошли в 2025 году===SELECT id, date, score, status FROM 	public.match WHERE status = 'ожидается' ORDER BY date;
Список игроков-белорусов, которые при этом не играют за клуб "Динамо Минск"===SELECT p.id, p.name, p.age, p.club_id, c.name AS club_name FROM public.player p JOIN public.club c ON p.club_id = c.id WHERE p.country = 'Беларусь' AND c.name <> 'Динамо Минск';
Список клубов у которых очков больше 80===SELECT c.id, c.name, cs.points FROM public.club c JOIN public.club_statistics cs ON c.id = cs.club_id WHERE cs.points > 80;
Список клубов со средним возрастом игроков более 25 лет===SELECT c.name FROM club c WHERE (SELECT AVG(p.age) FROM player p WHERE p.club_id = c.id) > 25;
Клуб с наибольшим количеством игроков===SELECT c.name FROM club c WHERE c.id = (SELECT club_id FROM player GROUP BY club_id ORDER BY COUNT(*) DESC LIMIT 1);
Список клубов с игроками из более чем двух разных стран===SELECT c.name FROM club c JOIN player p ON c.id = p.club_id GROUP BY c.name HAVING COUNT(DISTINCT p.country) > 2;
Список  клубов с количеством набранных очков выше среднего===SELECT c.name, cs.points FROM club c JOIN club_statistics cs ON c.id = cs.club_id WHERE cs.points > (SELECT AVG(points) FROM club_statistics);
Список игроков с максимальными очками в своем клубе===SELECT p.name, p.points, c.name AS club_name FROM player p JOIN club c ON p.club_id = c.id WHERE p.points = (SELECT MAX(points) FROM player WHERE club_id = p.club_id);
Список арен, на которых не проводились матчи===SELECT id, name FROM arena EXCEPT SELECT DISTINCT m.arena_id, a.name FROM match m JOIN arena a ON m.arena_id = a.id;
Список матчей с голом в овертайме и арбитром старше 40 лет===SELECT m.* FROM match m WHERE m.finish_type = 'овертайм' AND EXISTS (SELECT 1 FROM match_referee mr JOIN referee r ON mr.referee_id = r.id WHERE mr.match_id = m.id AND r.age > 40);
Список тренеров с опытом выше среднего===SELECT c.name, c.games_count FROM (SELECT AVG(games_count) AS avg_games FROM coach) AS avg_table, coach c WHERE c.games_count > avg_table.avg_games;
Список клубов, которые как выигрывали, так и проигрывали матчи===SELECT c.name FROM club c JOIN (SELECT club_id FROM club_match WHERE result = 'победа' INTERSECT SELECT club_id FROM club_match WHERE result = 'поражение') AS cl ON c.id = cl.club_id;
Количество игроков в каждом клубе, отсортировав по возрастанию===SELECT c.name AS club_name, p.club_id, COUNT(*) AS player_count FROM player p JOIN club c ON p.club_id = c.id GROUP BY c.name, p.club_id ORDER BY player_count ASC;
Общее количество матчей, сыгранных каждым клубом===SELECT c.name, COUNT(*) AS matches_played FROM club_match cm JOIN club c ON cm.club_id = c.id GROUP BY c.name;
Список игроков, у которых есть удаления, но нет забитых голов===SELECT p.name FROM player p WHERE EXISTS (SELECT 1 FROM player_match pm WHERE pm.player_id = p.id AND pm.penalties > 0 AND pm.goals = 0);
